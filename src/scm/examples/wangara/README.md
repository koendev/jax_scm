# Wangara Day 33 test case
Convective boundary layer test case based on the Wangara experiment (Day 33).
Initial conditions are radio sonde and balloon observations of 0900 local time.
Data are taken from the experiment report (Clarke, 1971).

**Reference**

> Clarke, R. H., et al. The Wangara Experiment: Boundary Layer Data. Technical Paper no. 19, 
> Commonwealth Scientific and Industrial Research Organtzation, 1971.
